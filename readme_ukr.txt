Українізація Об'єднаного Паку 2.
Версія 1.19.0 для ОП-2, патч 2.09, фікс 2

Установка:
1. ОП-2 (ТЧ, патч 1.0006, ОП-2)
2. Офіційний патч і фікс які підтримуються даною версією (див. зверху)
3. Українська озвучка (обережно, 157МБ) з архіву за посиланням https://www.dropbox.com/s/8pkj1gnfunct28z/official_ua_sounds.zip
3. Українізація.
4. Для того щоб у торгівельних діалогах і в рюкзаку валюта була "грн" замість "RU" замініть xrGame.dll з папки bin. Цей файл взято з релізу ОП-2. Брати звідсіля: https://www.dropbox.com/s/c3ss9gyulxynxjw/op2_xrgame_grn.zip
4.1 Якщо використовуєте інший xrGame.dll ніж оригінальний, то можете скористатись програмкою яка пропатчить Ваш xrGame.dll. Брати звідсіля: https://www.dropbox.com/s/fpljzxnfs3v0fv2/stalker_currency.zip
